Chatbot IA - Android (Java + ChatGPT)

Aplicación Android desarrollada en Java que integra un chatbot basado en inteligencia artificial utilizando la API de ChatGPT. Permite mantener conversaciones en lenguaje natural con respuestas en tiempo real.

Características
Chat en tiempo real con IA
Integración con la API de OpenAI (ChatGPT)
Manejo de contexto en conversaciones
Interfaz simple y fácil de usar
Respuestas rápidas
Tecnologías utilizadas
Java (Android)
Android Studio
OpenAI API (ChatGPT)
HTTP / REST APIs
JSON
